# Fortitude Bespoke Solutions & Risk Management - Website Project

## Project Overview

This project is the development and deployment of a premium security consultancy website for **Fortitude Bespoke Solutions & Risk Management**, a Dubai-based firm delivering high-level security advisory and risk management to ultra-high-net-worth individuals, family offices, and corporate executives globally.

**Website:** www.fortitudebsrm.com  
**Email:** inquiries@fortitudebsrm.com  
**Location:** Dubai, United Arab Emirates

---

## Brand Identity

### Core Positioning
Fortitude is a **consultancy-first, discreet, intelligence-led security firm**—not a guard supplier. The brand emphasizes:
- **Discretion** — Low-profile, confidential service delivery
- **Intelligence** — Founder-led expertise with Special Forces background
- **Precision** — Bespoke solutions tailored to individual threat profiles
- **Professionalism** — SIRA-certified, UK-qualified, regulatory compliant

### Aesthetic Direction
**Dark luxury, refined brutalism.** The site uses:
- Dark navy/charcoal backgrounds (#0f1419, #131822)
- Gold accents (#d4af37) for hierarchy and emphasis
- Minimal, clean typography (system fonts)
- No gradients, animations, or unnecessary effects
- Generous whitespace and precise spacing

**Tone:** Confident, understated, authoritative. No corporate jargon or cheerleading.

### Typography
- **Font Stack:** -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif (system fonts)
- **Headings:** Font-weight 700, slightly tighter letter-spacing
- **Body:** Font-size 0.95-1rem, line-height 1.8, color #b8bcc9
- **No decorative fonts** — keep it professional and clean

### Spacing & Layout
- Generous padding (2rem-3rem sections)
- Consistent gaps between elements
- Mobile-first responsive design
- Single-column layout below 768px

### Interactions
- Subtle hover states on cards (border color change, background shift)
- Smooth scroll behavior for navigation
- No animations or flashy effects
- Buttons with hover state: background/text color swap

## Technical Specifications

### File Provided
- **index.html** — Complete standalone website (no external dependencies)
- All CSS is inline in `<style>` tags
- All content is self-contained
- Responsive design for mobile, tablet, desktop

### Hosting Options
1. **Netlify** (recommended for ease)
   - Connect GitHub repo, auto-deploy on push
   - Free SSL, custom domain support
   - Drag-and-drop deployment

2. **Vercel**
   - Next.js-ready, but supports static HTML
   - Fast CDN, automatic deployments
   - Easy domain setup

3. **AWS S3 + CloudFront**
   - Scalable, secure
   - More technical setup
   - Good for enterprise

4. **GitHub Pages**
   - Free, simple
   - Good for portfolios
   - Push to main branch = live

5. **Traditional Hosting** (GoDaddy, Bluehost, HostGator)
   - Upload HTML file via FTP
   - Point domain via cPanel

### Domain Setup
- Purchase domain through registrar (Namecheap, GoDaddy, etc.)
- Point nameservers to hosting provider
- HTTPS should be automatic (most hosts provide free SSL)

### Performance
- Single HTML file (no build process needed)
- No external libraries or CDN calls
- Load time should be <1s on standard connections
- Optimized for Core Web Vitals (Lighthouse score: 90+)

### Contact Form Submission (Important)
The HTML form needs a backend service to send emails. Options:

1. **Netlify Forms** (easiest if hosted on Netlify)
   - Add `netlify` attribute to form: `<form name="contact" method="POST" netlify>`
   - Netlify handles email notifications automatically

2. **Formspree** (works anywhere)
   - Add action to form: `<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">`
   - Get form ID from formspree.io

3. **Backend Service** (custom)
   - Requires Node.js/Express server or similar
   - More complex setup
